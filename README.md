# NBTI 静态原型 v1

最简可测原型，支持：
- 首页
- 24 题答题流程
- 基于加权表计算结果
- 结果页第一屏
- 默认分享文案复制

## 本地预览

在当前目录运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173
```

## 文件说明

- `index.html`：页面入口
- `styles.css`：样式
- `app.js`：题目、人格数据、加权逻辑、页面交互
- `selftest.js`：基础脚本自测
